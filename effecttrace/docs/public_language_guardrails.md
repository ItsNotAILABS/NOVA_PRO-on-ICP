# EffectTrace Public Language Guardrails

## What EffectTrace Is

EffectTrace is a structured effect-tracing substrate for governance proposals on the Internet Computer.

It stores and renders what proposals claim to change, what the observable effect path is, what the runtime truth status is, and what questions remain unresolved.

## What EffectTrace Is Not

- EffectTrace does **not** recommend adopt or reject.
- EffectTrace is **not** an official DFINITY tool or standard.
- EffectTrace is **not** a replacement for human governance review.
- EffectTrace is **not** a replacement for CodeGov or any existing governance tooling.
- EffectTrace does **not** guarantee the accuracy of any proposal claim.

## Required Disclaimers

Every public-facing EffectTrace page must display:

> "This trace does not recommend adopt/reject. It is a structured effect record. Human verification required."

## Status Neutrality

Published ≠ Verified.  
ClaimOnly is a valid and useful status.  
Unknown is a valid and honest status.  
Disputed findings remain visible.

## No False Certainty

If the target canister, effect path, or verification state is unclear, it must be marked Unknown. Systems must not fabricate detail to fill gaps.
