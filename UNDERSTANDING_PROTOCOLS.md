## 🤔 What ARE Protocols?

### The Simple Answer

A **protocol** is a set of rules that defines how things communicate and work together.

Think of it like:
- **Language rules** - How words form sentences
- **Traffic rules** - How cars coordinate on roads
- **Dance choreography** - How dancers move together

### In Software

Protocols in software define:
1. **Format** - What data looks like
2. **Sequence** - Order of operations
3. **Behavior** - What happens when
4. **Standards** - Common agreements

**Examples you use every day:**
- **HTTP** - How web browsers get web pages
- **TCP/IP** - How internet packets travel
- **Bluetooth** - How devices connect wirelessly

### Example: What "Calls" and "Queries" Mean

When someone says **"protocols can call queries"**, they mean:

**In Regular Software:**
```javascript

  .then(response => response.json())  // ← This is a "query"
```

**In MEDINA:**
